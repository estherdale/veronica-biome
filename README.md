# Veronica-sexual-system-transitions-and-biome-shifts
Data and scripts for running analyses for Dale et al. Veronica seuxal system and biome shift paper

Veronica consensus.tree is the phylogeny used for all analyses
Veronica sexual system data.csv is data with sexual sytem data binary and categorical.
areas_allowed.txt file for time stratification with biome shift biogeograhic analysis with areas for each time period
full analysis.R script for the full analysis
Trait data from literature.csv is all the continuous data compile from the literature
time_periods.txt file for time stratification with biome shift biogeograhic analysis with timing of time periods
veronica_traits.csv sexual system and biome data formatted for biome dependent and independent biogeographic modelling
veronica_traits2.csv flower colour and biome data formatted for biome dependent and independent biogeographic modelling
